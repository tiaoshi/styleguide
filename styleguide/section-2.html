Here you can find more complex elements that rely on Javascript and Angular.

2.1Datepicker

The datepicker is based on bootstraps datepicker and all we do is wrap it and apply some special classes to make it more compatible with material design.

Example
From/to: 
<div class="wfm-datepicker-wrap">
<div class="sub-header">
	From/to:
</div>
<datepicker show-weeks="true" ng-model="someDate" class="wfm-datepicker"></datepicker>
</div >

2.2Date range picker

A directive for specifying a date-only period. It is composed from datepicker with added validation messages. In WFM, the error messages will be translated automatically.

Attribute NameUsageRequired
start-dateBind to period start datetrue
end-dateBind to period end datetrue
errorsBind to errors. Its value will be set by the picker to an array of error-keys. The pre-defined error keys are 'order' and 'empty'false
hide-messagemute the error messagefalse
Example 
<date-range-picker start-date="startDate" end-date="endDate"></date-range-picker>

2.3Line chart

A chart is a great way to present complex data. We use C3 charts to get nice features like legends, zoom and period selection.

Javascript example
var chart = c3.generate({
   bindto: '#myChart',
   data: {
       columns: [
        ['data1', 30, 200, 100, 400, 150, 250],
           ['data2', 20, 180, 240, 100, 190,0]
       ],
       selection: {
   enabled: true
 }
   },
   subchart: {
       show: true
   },
zoom: {
       enabled: true
   }
});

Example 
<div class="wfm-chart" id="myChart"></div>

2.4Card List

The Teleopti Card List is based on the card list displayed in google inbox. Each card can be clicked on within the list and will expand to show detail.
Only one card can be opened at a time.The header and content of each card is completely customizable and any html elements provided within the <card-header> and <card-body> tags will be rendered appropriately.

Example
multi-select 
<wfm-card-list multi-select>
<wfm-card ng-repeat="item in items">
	<card-header>
		<span><i class="mdi" ng-class="item.title"></i></span>
		multi-select
		<span class="chevron"><i class="mdi mdi-chevron-right"></i></span>
	</card-header>
	<card-body>
		<button class="wfm-fab info material-depth-1"><i class="mdi mdi-duck"></i></button>
	</card-body>
</wfm-card>
</wfm-card-list>
